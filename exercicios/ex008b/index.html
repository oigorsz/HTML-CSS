Código Fonte

Para transcrever um código fonte em HTML (efeito monoespaçado), é necessário usar a tag <pre> + <code> 


    let a = document.getElementById("a")
    if (a.value > 10){
        window.alert("maior que 10")
    } else {
        window.alert("menor que 10")
    }

Citações

Para realizar uma citação, utliza-se a tag <q> ou a <blockquote cite="link da onde tirou a citação">

Ou também...


Newton disse:
A força é igual a massa multiplicada pela aceleração

Abreviações

Para informar o que siginifica determinada nomeclatura, pode-se usar a tag <abbr>.
